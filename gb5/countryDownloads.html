Individual Country Files

The geoBoundaries open license (CC-BY 4.0) country dataset seeks to represent every nation "as they would represent themselves", with no special identification of disputed areas. If you would prefer data that explicitly includes disputed areas, please see the global composite dataset. The only requirement for use of this dataset is attribution.
